Pipeline:

1. Obtain list of training PMIDs

   python3 ImportAnnotatedData.py

2. Retrieve abstracts in PubTator format with GNormPLus in perl and annotate entities with GNormPlus in java

   cd ../GNormPlusPerl
   bash retrieve_training_pmids.sh
   
   cd ../GNormPlusJava
   bash annotate_pmids.sh
   
3. Build training set with replaced entities

   python3 ExportAnnotatedDataWithEntities.py
    
4. Preprocess training set

   python3 PreprocessAnnotatedData.py
    
5. Build preprocessed training sets for Machine Learning

   python3 BuildTrainingData.py
   
6. Run BPE

    bash BPE.sh

I. A. Titles + abstract
   B. Titles + abstract (with GNorm entitites)
   
II. A. Abstract
   B. Abstract (with GNorm entitites)
