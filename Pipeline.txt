Pipeline:

1. Obtain list of training PMIDs

   python3 ImportTrainingData.py

2. Retrieve abstracts in PubTator format with GNormPLus in perl and annotate entities with GNormPlus in java

   cd ../GNormPlusPerl
   bash retrieve_training_pmids.sh
   
   cd ../GNormPlusJava
   bash annotate_training_pmids.sh
   
3. Build training set with replaced entities.

    python3 ExportTrainingDataWithEntities.py
